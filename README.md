# Statistical-analysis-of-customers-spending
The aim of this paper is to analyze how the customer expenditure vary with respect to different factor, like age, gender and income, in order to make a model for predicting the customerâ€™s expenditure. The data are collected by an American survey run outside of a supermarket. The analysis is done by running some bivariate test, association and correlation test in order to do a complete linear regression for correctly understand the implication of some factors on customer spending. After that we will discuss the fitting of a regression tree for the same purpose.
